
# Finance Management Website

A modern personal finance management web app built with React, TypeScript, and Tailwind CSS. Track your income and expenses, monitor your budget, visualize spending trends, and get smart financial insights.

## Features

- Dashboard with financial overview and health score
- Expense & income tracking with transaction history
- Budget tracker with category breakdowns
- AI-powered financial insights and chat assistant
- Future financial simulator
- Voice assistant support
- Responsive, dark-mode-ready UI

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** for fast dev/build
- **Tailwind CSS** for styling
- **shadcn/ui** component library
- **Recharts** for charts and visualizations
- **React Router** for navigation

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
